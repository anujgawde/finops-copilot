# Ask Pump: AI FinOps Copilot

A conversational cloud cost intelligence interface that lets engineering leads, finance teams, and founders ask plain-English questions about their AWS spend and get structured answers: root-cause analysis, inline charts, savings opportunities, and board-ready exec summaries.

---

## Tech stack

| Layer | Choice |
|---|---|
| Framework | Next.js 14 (App Router) |
| Language | TypeScript (strict) |
| Styling | Tailwind CSS |
| Animation | Framer Motion |
| Charts | Recharts |
| Icons | lucide-react |
| Dialog | @radix-ui/react-dialog |
| Utilities | clsx |

---

## Getting started

```bash
# Clone the repo
git clone https://github.com/your-username/finops-copilot.git
cd finops-copilot

# Install dependencies
npm install

# Start the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

### Verify the build

```bash
npm run typecheck   # must be clean
npm run build       # must succeed
```
